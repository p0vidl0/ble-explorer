# Devices (Standalone)

Независимый модуль страницы BLE-устройств, вынесенный в отдельный каталог для последующего переноса в отдельный репозиторий.

## Запуск

```bash
pnpm install
pnpm start
```

## Сборка

```bash
pnpm build
```

## Особенности

- не зависит от корневого `src/` приложения;
- содержит собственный BLE-модуль (`src/lib/web-ble.js`);
- имеет отдельный `package.json` и свои скрипты.
